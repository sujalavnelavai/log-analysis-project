# Beginner Log Analysis Project

This project demonstrates a simple SOC-style log analysis exercise.  
It includes a sample log file and a detailed analysis of suspicious activity.

## Skills Demonstrated
- Log analysis
- Threat detection
- Pattern recognition
- Basic incident response
- Security investigation mindset

## Files
- sample-log.txt — raw log data
- analysis.md — findings and recommendations

## Purpose
To practice beginner-friendly SOC Analyst skills and build a cybersecurity portfolio.
